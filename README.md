# Hole-e-Grail

## Project Overview
**Hole-e-Grail** is a web-based pothole detection and alert system that uses real-time GPS tracking to help users avoid potholes on roads.

The system displays pothole locations on an interactive map and alerts users when they are within 50 meters of a pothole.

## Features
- Interactive map visualization  
- Real-time user location tracking  
- Pothole severity markers  
- Automatic proximity alerts  
- Flask backend API  
- Data fetching using Fetch API  

## Technology Stack
- Backend: Flask (Python)  
- Frontend: HTML, JavaScript, Leaflet.js  
- API Communication: REST API  
- Styling: CSS
